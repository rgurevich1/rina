# Marina Gurevich — Portfolio

Personal portfolio site for **Marina Gurevich**, a Graphic Designer specializing in Brand Identity, UX/UI Design, and Digital Multimedia Design.

## About

A multi-page portfolio built as self-contained, offline-ready HTML files. Each page embeds its own fonts and images — no build step, no dependencies, no internet required. Just open a file in a browser.

## Pages

| File | Page |
|------|------|
| `index.html` | About (landing page) |
| `About.html` | About |
| `Work.html` | Selected work — podcast cover artwork, brand identity, storytelling |
| `Services.html` | What I do |
| `Experience.html` | Experience & education |
| `Skills.html` | Skills |
| `Contact.html` | Contact |

The top navigation links the pages together. The site is responsive and works on mobile.

## View locally

Open `index.html` in any web browser.

## Publish with GitHub Pages

1. Upload all the files to the **root** of a GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Branch**, choose `main` and `/ (root)`, then **Save**.
4. After a minute, your site is live at:
   `https://<your-username>.github.io/<repo-name>/`

`index.html` is served automatically as the home page.
