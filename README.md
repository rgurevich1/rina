## How it's built

Plain HTML pages with a small runtime (`support.js`) and an `images/` folder. No build step and no in-browser decompression, so it works on every browser and device (including older iPad Safari).

## Publish with GitHub Pages

1. Upload **all** files and the `images/` folder to the repository **root**.
2. Settings → Pages → Branch `main`, folder `/ (root)` → Save.
3. Live at `https://<username>.github.io/<repo>/`.
